# Writing Guide with Handbook

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/openstax/osbooks-writing-guide)

_Writing Guide with Handbook_ is a textbook published by [OpenStax](https://openstax.org/), a non profit organization that is part of [Rice University](https://www.rice.edu/).

The book can be viewed [online](https://openstax.org/details/books/writing-guide), where you can also see a list of contributors.
